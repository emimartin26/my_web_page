Sobre mí

Me llamo Emiliano Martin, soy estudiante de ingeniería en sistemas de información, en la Universidad Tecnológica Nacional Facultad Regional Villa María(Argentina). Actualmente me encuentro terminando mi proyecto de grado, para obtener el titulo de ingeniero.

Me apasiona el desarrollo de software que implique la participación de múltiples tecnologías ya sea web, mobile, desktop.

Durante el cursado de mi carrera me especialice en el lenguaje de programación Java y el motor de bases de datos postgresql.

Paralelo a esto, comencé a descubrir mi pasión por el mundo web y me embarque en el aprendizaje autodidacta, aprendí las bases del lenguaje python y el uso del framework de desarrollo web Django. Y en la actualidad estoy aprendiendo a crear aplicaciones MEAN(mongodb, express, angular.js, node.js), ya que considero que el presente y el futuro de la web esta fuertemente marcado por el lenguaje JavaScript.

Dentro del mundo web, me gusta y me especializo mas en la parte del Back-End(Django, Node.Js) pero también he ido adquiriendo conocimientos en el Front-End(Html, Css, JavaScript, Bootstrap, Angular.Js).

En 2014 obtuve el Premio a la Excelencia Académica otorgado por Intel ASDC.

También participe en un grupo de investigación dentro de la Universidad. Dicha investigación fue presentada en conaiisi 2014 – San Luis – Argentina. La investigación se titulo: "Testing Ágil - Presente y Futuro".

Me gusta cada día aprender nuevas herramientas, lenguajes, metodologías de desarrollo y estar al día sobre las ultimas tendencias del mundo del software.

Si te interesa lo que hago ponte en contacto conmigo, Saludos.

IRIS

IRIS (Interactive Restaurant Information System) es un proyecto de final de grado, que estamos desarrollando junto con cuatro compañeros de la universidad para obtener el titulo de ingenieros en sistemas de información. El sistema permite gestionar platos, promociones , mesas, atención de clientes, pedidos, empleados, usuarios como así también obtener información del estado de los pedidos en tiempo real. Para este proyecto se están utilizando múltiples tecnologías.
Para el trabajo CRUD(Create, Read, Update and Delete) se utiliza Django como framework back-end, como motor de bases de datos se utiliza PostgreSql. Como lenguaje de front-end se utiliza Html, Css y JavaScript, haciendo uso del framework Bootstrap. Para los dispositivos móviles encargados de tomar pedidos, se utiliza Lungo.JS. Y por ultimo para la comunicación en tiempo real, se utiliza Node Js y Socket.IO.

  • Cliente: Restaurante Chúcaro
  • Fecha: Marzo 2014 - Actualidad
  • Tecnologías: Html5 - Css3 - JavaScript - Bootstrap - Lungo.JS - Node.JS - Socket.IO
